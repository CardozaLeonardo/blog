Introducción

CSS, las siglas para Cascading Style Sheets, u Hojas de estilo en cascada, es una tecnología que nos permite crear una página web de manera más exacta; o, siendo más especfícicos, que nos permite editar una página web.

Si bien se puede decir que HTML (se recomienda leer antes esa entrada del blog) es la estructura de una página web, CSS es la forma de darle ese atractivo visual a la página, permitiéndonos ordenar los textos, darles color, poner un fondo, crear banners, etc., en fin, todo para una edición de página web.

Ahora, ¿cómo se trabaja con CSS? Simple. Si bien es recomendable utilizar un editor de texto más "profesional", crear hojas de estilo con CSS (cabe recalcar que CSS es una forma de trabajar hojas de estilo, pero existen más) con el simple bloc de notas que traen todas las computadoras.

La base, y lo primordial, es crear un archivo con extensión ".css". Luego de esto, ya podemos comenzar a editar nuestra hoja de estilo.
